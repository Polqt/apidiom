apidiom turns an OpenAPI file, URL, or built-in service name into agent-ready
tools.

- Generate a standalone JavaScript MCP server for MCP-compatible clients.
- Export OpenAI or Anthropic JSON tool schemas for raw SDK use.
- Filter large APIs with `--tag` or `--include` before generating output.

Start with the [quickstart](/quickstart/).
