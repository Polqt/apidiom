title: Troubleshooting
description: Common errors and what to do next.
---

## Source Not Found

Use `./` for local files when a path could be confused with a built-in service
name.

```bash
apidiom generate mcp ./openapi.yaml --output api-mcp.js
```

## Missing Source

`generate mcp` requires a source unless you are listing built-in services.

```bash
apidiom generate mcp --list
apidiom generate mcp stripe --output stripe-mcp.js
```

## Missing Schema Format

Raw SDK schema export requires an explicit format.

```bash
apidiom generate schema ./openapi.yaml --format openai
apidiom generate schema ./openapi.yaml --format anthropic
```

If the format is not `openai` or `anthropic`, apidiom exits with a clear error.

## Too Many Tools

Very large specs can produce hundreds of tools. Filter first:

```bash
apidiom generate mcp github --tag repos --output github-repos-mcp.js
apidiom generate schema stripe --format openai --include createPaymentIntent
```

Use `--tag` for broad resource groups. Use `--include` when you know the exact
operation IDs.

## Runtime Auth Error

Generated MCP servers read supported credentials from environment variables. If
a required value is missing, the generated server prints the missing variable
name and exits.
