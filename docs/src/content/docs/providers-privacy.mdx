This page is archived. The current TypeScript CLI does not call LLM providers
and does not send specs to paid services.

Current generated outputs are:

- `apidiom generate mcp <source>` for standalone MCP servers.
- `apidiom generate schema <source> --format openai|anthropic` for raw SDK
  tool schemas.

Use [Quickstart](/quickstart/) for current commands.
